Cosmic Cafe

A responsive cafe website built with pure HTML, CSS, and JavaScript. Features a warm cosmic-inspired palette, smooth UI, menu filtering, gallery lightbox, dark mode, and a simple cart prototype.

Features

Mobile-first responsive layout

Custom color palette (CSS variables)

Hero with subtle star animation

Menu filtering (JS)

Gallery with lightbox

Dark mode with localStorage

Cart modal prototype

Basic contact form validation
